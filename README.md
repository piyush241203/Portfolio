# Piyush Ninawe - Portfolio

This is a personal portfolio website built with Next.js, React, and Tailwind CSS.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Stack
- **Framework:** Next.js
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Icons:** Lucide React & React Icons

## Deployment (Vercel)

This project is configured and ready to be deployed on Vercel. Vercel will automatically detect that this is a Next.js application.

1. Push your code to a GitHub repository.
2. Go to [Vercel](https://vercel.com/) and sign in.
3. Click "Add New..." -> "Project".
4. Import your GitHub repository.
5. Keep the default settings (Vercel automatically sets the build command to `next build` and install command to `npm install`).
6. Click "Deploy".

Your portfolio will be live in a few minutes!
